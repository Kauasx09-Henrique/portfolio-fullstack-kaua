# KAUÃ HENRIQUE ✦ PORTFOLIO

> A masterclass in minimalist web design, software architecture, and performance.

Este repositório contém o código-fonte do meu portfólio pessoal. Desenvolvido sob uma rigorosa estética *Avant-Garde Editorial*, o projeto une tipografia de ponta, micro-interações a 60fps e integrações de dados em tempo real para criar uma experiência de navegação fluida e de alto impacto para recrutadores e tech leads.

---

## ✦ A Arquitetura Visual

O design system baseia-se no conceito de **"Zebra Striping Editorial"**, alternando sequencialmente entre blocos de branco puro (`#ffffff`) e preto absoluto (`#000000`). Esse ritmo visual de respiração e contraste permite que cada seção do portfólio brilhe individualmente, guiando a leitura sem sobrecarregar a carga cognitiva do usuário.

### Estrutura do Projeto (Sections)

1. **Home:** O primeiro impacto, focado em tipografia colossal e apresentação limpa.
2. **About:** Um mergulho no meu perfil técnico e visão de engenharia.
3. **Trajetória (Journey):** Linha do tempo vertical destacando minha atuação no SENSIPAM (Centro Operacional e Gestor da Amazônia) e meu avanço no 5º semestre de Análise e Desenvolvimento de Sistemas.
4. **Tech:** Grid geométrico apresentando o domínio do ecossistema JavaScript/TypeScript.
5. **Projetos:** Exibição detalhada de arquiteturas complexas (Full Stack & Mobile).
6. **Métricas (Telemetry):** Integração com a API do GitHub para exibir dados de commits e repositórios em tempo real.
7. **Certificados:** Validação acadêmica e cursos extracurriculares.
8. **Currículos (Dual-Profile):** Sistema de modais nativos para visualização de currículos focados em *Desenvolvimento Full Stack* ou *Suporte Técnico N1/N2*.
9. **Contato:** Encerramento limpo para geração de leads e parcerias.
10. **Footer (Monolith):** Rodapé em preto absoluto finalizando a estrutura com um botão interativo de retorno ao topo.

## ✦ Tech Stack & Engenharia

A infraestrutura foi montada visando máxima performance e zero dependências desnecessárias, refletindo meu foco em arquiteturas escaláveis.

| Camada | Tecnologia | Propósito |
| :--- | :--- | :--- |
| **Core** | `React (Vite)` | Renderização ultrarrápida e build otimizado para produção. |
| **Styling** | `CSS Modules / Vanilla` | Controle absoluto de propriedades sem o overhead de frameworks visuais. |
| **Motion** | `Framer Motion` | Coreografia de animações físicas (springs, dampings e efeitos de glitch). |
| **Icons** | `Lucide React` | Iconografia vetorial limpa e escalável para telas retina. |
| **Data** | `Fetch API` | Consumo assíncrono e leve de dados externos (GitHub REST API). |

## ✦ Projetos em Exibição

O portfólio atua como um hub central para as soluções de alto nível que desenvolvi e estruturei:

* **Marcador:** App Mobile Full Stack construído com React Native, Node.js e PostgreSQL. Focado em leitura social, gamificação, sincronização real-time (Modo Duo) e *Smart Fallback* de APIs para escaneamento de ISBN.
* **Bibliafy:** Plataforma completa para leitura bíblica estruturada em React com API própria.
* **ClinicPed+:** Sistema modular (Mobile + API) de gestão para clínicas médicas, utilizando NestJS para um backend robusto e seguro.
* **DisBusiness & System Barber:** Aplicações de negócio focadas em painéis administrativos, UX e processos eficientes de banco de dados.
